# 🎮 Futuristic Gaming Website (Cyberpunk UI)

A visually stunning **cyberpunk-inspired gaming website** built using **HTML, CSS, and JavaScript**.
This project focuses on **modern UI/UX**, **neon aesthetics**, and **immersive animations** to deliver a next-level user experience.

---

## 🚀 Features

### 🌌 Design & UI

* Dark **cyberpunk theme** with neon (green, blue, purple) glow
* Futuristic typography and layout
* Glassmorphism + glowing UI components
* Fully responsive across all devices

---

### 🎯 Core Sections

* 🦾 **Hero Section**

  * Game title with **glitch text effect**
  * Animated CTA button
  * Dynamic background

* ⚡ **Game Features**

  * Icon-based feature cards
  * Hover animations and glow effects

* 🖼️ **Screenshots Gallery**

  * Interactive image grid
  * Hover zoom effects

* 🎬 **Trailer Section**

  * Embedded gameplay video
  * Smooth reveal animations

* 🌐 **Community Section**

  * Social links and engagement area
  * Call-to-action for joining community

---

### ✨ Interactions & Animations

* 🔥 Glowing buttons with hover effects
* 🌊 Smooth scroll animations
* 🎇 Animated particle background
* ⚡ Micro-interactions for better UX
* 👾 Optional sound effects on hover/click

---

### ⏳ Extra Features

* 🌀 **Loading Screen Animation**

  * Futuristic loader before website loads

* 💥 **Glitch Effects**

  * Cyberpunk-style glitch text animations

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Flexbox, Grid, Animations)**
* **JavaScript (Vanilla JS)**
